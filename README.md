## Technology Stack

### Core Framework
- **Framework**: Next.js 15 with App Router and Turbopack
- **UI Library**: React 19 with TypeScript
- **Styling**: Tailwind CSS v4 with design tokens
- **Components**: shadcn/ui (New York variant) with Radix UI primitives
- **Theming**: next-themes with automatic dark mode

### Animation & Motion
- **motion** v12.23.24 - Framer Motion for React animations
- **tw-animate-css** v1.3.8 - Tailwind-compatible animation utilities
